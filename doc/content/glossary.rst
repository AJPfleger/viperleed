.. _glossary:

========
Glossary
========

.. glossary::

    LEED
      Low energy electron diffraction

    DFT
      Density functional theory

    TensErLEED
      Erlangen package for Tensor LEED

    ViPErLEED
      The Vienna Package for TensErLEED

    VASP
      Vienna ab initio simulation package (`VASP wiki <https://www.vasp.at/wiki/index.php>`__)

    VESTA
      3D Visualization software for structural models (`Homepage <https://jp-minerals.org/vesta/en/>`__)

    LEED-I(V)
      Energy dependent low energy electron diffraction.

    GUI
      Graphical User Interface

    MPI
      Message Passing Interface. (`Wikipedia entry <https://en.wikipedia.org/wiki/Message_Passing_Interface>`__)

    tleedm
      The TensErLEED manager, the Python module of ViPErLEED.

    Python
      The Python programming language. (`Homepage <https://www.python.org>`__)
